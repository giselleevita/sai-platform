## 💼 Where to Place in CV

### If Recent/Current Project:
Place in **"Projects"** or **"Experience"** section near the top

### If Portfolio Project:
Include in **"Portfolio Projects"** or **"Side Projects"** section

### Format Suggestions:

1. **Projects Section** (if not employed):
   ```
   PROJECTS
   └── SAI Platform (Full-Stack Developer)
   ```

2. **Experience Section** (if freelance/contract):
   ```
   EXPERIENCE
   └── SAI Platform - Full-Stack Developer
   ```

3. **Technical Projects** (if employed elsewhere):
   ```
   TECHNICAL PROJECTS
   └── SAI Platform
   ```

## 💡 Tips

1. **Tailor to Job**: Emphasize technologies/features relevant to the role you're applying for
2. **Be Specific**: Use numbers (29 features, 80+ endpoints) rather than vague terms
3. **Show Impact**: Mention business value (compliance, risk management, audit readiness)
4. **Keep It Concise**: CV should be scannable - use bullet points, not paragraphs
5. **Update Regularly**: Keep project description current as you add features
